10 Design Patterns

다마고치 공식 사이트에서 추출한 핵심 디자인 패턴 요약입니다.

Core Principles

PrincipleDescription
White Canvas흰 배경 위에 다채로운 이미지/일러스트로 활기를 부여. CSS 색상은 절제하고, 컨텐츠가 색을 제공.
Bold Borders3px 네이비 보더로 카드와 UI 요소에 장난감 같은 느낌을 부여.
Pill Shapes버튼과 뱃지는 완전한 라운드(pill) 형태. 카드는 16-24px radius.
Bilingual Headers영문 대문자 + 일본어/한국어 서브타이틀의 이중 구조.
Section Alternation스카이 블루, 옐로우, 화이트, 네이비 배경을 번갈아 사용.
Kawaii-Retro Fusion레트로 픽셀아트 + 현대적 카와이 일러스트의 조합.
Generous Whitespace섹션 간 80px 이상의 충분한 여백으로 시각적 여유를 확보.
Playful Motion호버 시 부드러운 상승(translateY) + 바운스 이징.
